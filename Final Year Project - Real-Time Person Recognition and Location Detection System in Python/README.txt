In PHP codes:
Load the front-end using WAMP server, which uses index.php 

In Python Codes:
modules to install.txt lists the Python modules needed and the order in which they should be installed.
Use datasets.py to create a facial images dataset.
Use encode_faces.py to train the CNN on the dataset.
Use train_model to train a CNN on MNIST dataset.
Run the application by executing pi_face_recognition.py first and then submitting a name in the front-end.
